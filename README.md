# Guess-o-Matic
🎮 A fun and interactive "Guess the Number" desktop game built with Python and Tkinter! 🐍 Test your luck and logic by guessing a secret number between 1 and 100. 🔢 The app provides helpful "Higher ⬆️" or "Lower ⬇️" hints until you crack the code. Can you find the exact number in the fewest attempts possible? 🏆
