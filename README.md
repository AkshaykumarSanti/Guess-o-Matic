Guess-o-Matic💡

A fun and interactive Python GUI game built using 🐍 Tkinter where the player tries to guess a randomly generated number between 1 and 100.

Test your intuition, improve your logical thinking, and enjoy a clean desktop interface! 💡✨

🕹️ How The Game Works

The system generates a random number between 1 and 100 🎲

You enter your guess in the input box

The game gives you hints:

🔺 Higher Number Please

🔻 Lower Number Please

When you guess correctly:

✅ It shows the number of attempts

🔒 The submit button gets disabled

Simple. Clean. Addictive. 😎

🖥️ Tech Stack Used

🐍 Python

🖼️ Tkinter (GUI Library)

🎲 Random Module

🚀 Features

✨ User-friendly GUI
⌨️ Supports Enter key submission
🔢 Input validation (handles invalid entries)
📊 Tracks number of attempts
🎨 Color-coded feedback messages
🔒 Disables button after correct guess
